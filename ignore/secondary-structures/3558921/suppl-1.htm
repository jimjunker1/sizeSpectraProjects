Description

Demonstration of the predictions of generalized MTE model can be carried out by executing the demo function, SI_demonstration.r, and then loading the example data sets, ssize.csv and sizeTL.csv, and prediction functions (Prediction_func.r and Random_Test.r) following the instructions in the demo function. The demon function, SI_demonstration.r, is the main function to demonstrate the implementation of two predictions from the example data sets and calculation of the 95% confidence interval. Prediction_func.r is the functions that produces the cross predictions between ISD and size-TL relationship (ISD→STL and STL→ISD) in the generalized MTE model. Random_Test.r is the function for the randomization test of the generalized MTE model.
